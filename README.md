# 💰 Finance Tracker

A personal finance dashboard built with Next.js, TypeScript, and Tailwind CSS.

## Features

- 📊 Dashboard with income vs expense charts
- 💳 Transaction management (add, edit, delete)
- 🌙 Dark/Light mode toggle
- 📱 Fully responsive

## Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS v4
- Recharts
- Lucide React
